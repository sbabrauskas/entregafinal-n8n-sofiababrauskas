# AI Job Relevance Automation Workflow

## Requisitos
- n8n instalado
- Cuenta OpenAI y Api Key
- Cuenta Gmail conectada en n8n

## Cómo importar
1. Ir a n8n
2. Import workflow.json
3. Configurar credenciales OpenAI
4. Configurar credenciales Gmail
5. Activar workflow

## Cómo probar
Esperar ejecución automática o ajustar Schedule a 1 minuto.
